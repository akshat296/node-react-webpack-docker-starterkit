# node-react-webpack-docker-starterkit

A simple Nodejs and React started kit that can be used to bootstrap your new React application easily.

## Features and Modules
1. React
2. Webpack2
3. Babel
4. Docker file
5. pm2
6. Enzyme
7. Chai
8. Semantic-UI
9. Pug Template engine
